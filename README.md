# SHL Assessment Recommendation Engine

This project builds a web-based recommendation engine that suggests relevant SHL Individual Test Solutions based on a natural language job requirement.

## Features
- Crawls SHL Product Catalogue
- Filters Individual Test Solutions
- Keyword-based NLP matching
- Streamlit web interface

## Tech Stack
- Python
- Requests & BeautifulSoup
- NLTK
- Scikit-learn
- Streamlit
